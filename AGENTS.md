## Project Overview

- **License:** GPL-3.0

## Architecture

MCP (Model Context Protocol) server built in Go. Exposes tools and resources for AI assistants to interact with Pumperly functionality.

### Key Files

- `main.go` — Entry point, MCP server initialization
- `tools/` — MCP tool implementations
- `Dockerfile` — Container build
